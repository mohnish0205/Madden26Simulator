# Madden26Simulator
Madden Football '26 is a Python project designed to recreate the excitement of NFL football directly in the terminal. Inspired by the Madden franchise, this simulator combines strategic play-calling, realistic football mechanics, team ratings, and dynamic game outcomes to deliver an immersive football experience without requiring any graphics engine.

Built with a focus on realism, replayability, and presentation, the game allows players to select from all 32 NFL teams, compete against the computer, and experience a complete football game from kickoff to the final whistle. Team overall ratings influence performance, while offensive and defensive play choices create unique situations every game.
Features

NFL Team Selection: Choose from all NFL teams, each with unique overall ratings that affect gameplay performance. Team ratings influence offensive production, defensive success, kicking ability, and overall game outcomes.

Interactive Play Calling: Take control of your team's offense and defense by selecting plays throughout the game. Choose between runs, short passes, deep passes, blitzes, zone coverage, and more to outsmart the CPU opponent.

Realistic Game Simulation: Features downs, distance tracking, first downs, touchdowns, field goals, punts, turnovers, safeties, possession changes, and game clock management. Every play affects field position and strategy.

AI Decision Making: The CPU dynamically selects offensive and defensive plays based on game situations, including fourth-down decisions, field goal attempts, and punting strategy.

Advanced Statistics Tracking: The simulator records rushing yards, passing yards, total offense, turnovers, sacks, time of possession, and other key football statistics to create a complete post-game analysis.

Enhanced Terminal Interface: Uses colorful ANSI graphics, scoreboards, field displays, play logs, menus, and custom box-drawing interfaces to create a visually engaging experience entirely within the command line.

Scoring & Special Teams: Includes touchdowns, extra points, two-point conversions, field goals, kickoffs, and punts, allowing players to experience every phase of football.

High Replayability: Because outcomes are influenced by team ratings, AI decisions, random game events, and player strategy, no two games play out exactly the same.

Goal of the Project
The goal of Madden Football '26 is to demonstrate how a full football simulation can be built using only Python and terminal technology. By combining strategy, randomness, and NFL-inspired gameplay mechanics, the project delivers an entertaining football experience while showcasing programming concepts such as game logic, state management, user interaction, and statistical simulation.
